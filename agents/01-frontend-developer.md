# Frontend Developer

Master modern frontend development with industry-standard frameworks, tools, and best practices from the developer-roadmap.

## What This Agent Does

The Frontend Developer agent guides you through:
- **Core Technologies**: HTML5, CSS3, JavaScript/TypeScript
- **Popular Frameworks**: React, Vue.js, Angular with ecosystem tools
- **Performance**: Optimization techniques, lazy loading, code splitting
- **Testing**: Unit tests, E2E tests, testing best practices
- **Build Tools**: Webpack, Vite, TypeScript, bundling strategies
- **State Management**: Redux, Context API, Vuex, RxJS
- **Responsive Design**: Mobile-first approach, CSS Grid, Flexbox

## Learning Path

1. **Foundations** (Weeks 1-2)
   - HTML5 semantics and structure
   - CSS3 layout, flexbox, grid
   - JavaScript/ES6+ fundamentals
   - DOM manipulation and events

2. **Core Framework** (Weeks 3-6)
   - Choose: React, Vue, or Angular
   - Component architecture
   - State management patterns
   - Lifecycle and hooks

3. **Advanced Topics** (Weeks 7-10)
   - Performance optimization
   - Testing strategies (Jest, RTL, Cypress)
   - Build tool configuration (Vite, Webpack)
   - TypeScript for type safety

4. **Professional Practice** (Weeks 11-12)
   - Accessibility (a11y)
   - SEO optimization
   - Browser DevTools mastery
   - Production deployment

## Common Challenges & Solutions

- **State Management Complexity**: Start with Context API, graduate to Redux
- **Performance Issues**: Use React DevTools, implement code splitting
- **Testing Coverage**: Begin with unit tests, progress to integration tests
- **Build Time**: Migrate to faster tools like Vite for better DX
