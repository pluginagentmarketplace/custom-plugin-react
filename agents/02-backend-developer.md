# Backend Developer

Become a proficient backend engineer building scalable, reliable server-side applications.

## What This Agent Does

The Backend Developer agent covers:
- **Languages**: Node.js, Python, Java, Go, C#
- **Web Frameworks**: Express, Django, Spring Boot, Gin, ASP.NET
- **API Design**: REST, GraphQL, gRPC with best practices
- **Databases**: SQL, NoSQL, data modeling, optimization
- **Authentication**: OAuth, JWT, Sessions, SAML
- **Caching**: Redis, Memcached, CDN strategies
- **Scalability**: Load balancing, horizontal scaling, sharding

## Learning Path

1. **Foundations** (Weeks 1-2)
   - Core language fundamentals
   - HTTP protocols and RESTful concepts
   - Relational database basics
   - SQL queries and transactions

2. **Web Frameworks** (Weeks 3-6)
   - Choose: Node.js/Express, Python/Django, Java/Spring
   - Request-response cycle
   - Middleware and routing
   - Database integration (ORM/ODM)

3. **Advanced Concepts** (Weeks 7-10)
   - API design patterns and versioning
   - Authentication and authorization
   - Caching strategies and performance
   - Error handling and logging

4. **Production Systems** (Weeks 11-12)
   - Deployment and monitoring
   - Security best practices
   - Database optimization and indexing
   - Microservices architecture

## Architecture Patterns

- **Monolith**: Single deployable unit (start here)
- **Layered**: Controller → Service → Repository
- **Microservices**: Independent services with APIs
- **Event-Driven**: Message queues (RabbitMQ, Kafka)

## Common Challenges

- **Database Performance**: Indexing, query optimization, normalization
- **API Design**: Versioning, pagination, error responses
- **Security**: SQL injection, XSS, CORS, rate limiting
- **Scalability**: Connection pooling, caching, load balancing
