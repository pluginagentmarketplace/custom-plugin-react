## üõ†Ô∏è Troubleshooting

### Command Not Working

**Problem**: `/learn-path` doesn't respond

**Solution**:
1. Verify plugin is loaded: `/plugin list`
2. Try again: `/learn-path`
3. Check Claude Code docs: `/help plugin`

---

### Agent Not Responding

**Problem**: `@frontend-developer` doesn't reply

**Solution**:
1. Use correct format: `@specialization-name question`
2. Ask specific question, not vague request
3. Provide context about your situation
4. Try simpler question first

---

### Resources Not Loading

**Problem**: `/find-resources` returns outdated info

**Solution**:
1. Plugin updates resources every 24 hours
2. Force refresh by running command twice
3. Check official docs directly
4. Report issue on GitHub

---

### Plugin Installation Issues

**Problem**: Plugin won't load

**Solution**:
```bash
# Ensure correct directory
ls ~/.claude-code/plugins/

# Verify plugin.json exists
cat .claude-plugin/plugin.json

# Check Claude Code version
claude --version

# Try local loading
```

## ‚ùì FAQ

### Q: How long does each path take?
**A**: 12 weeks at 30-40 hours/week, or 6 months at 15-20 hours/week. Adjust based on your pace.

### Q: Can I do multiple paths?
**A**: Yes! Many combine paths (Frontend + Backend for full stack). See `/explore-roadmap` for combinations.

### Q: Are there project requirements?
**A**: No, but building real projects transforms learning. We provide project ideas at each level.

### Q: How do I get a job after?
**A**: Build portfolio, contribute to open source, attend meetups, practice interviews. We guide through this.

### Q: What if I'm switching careers?
**A**: Many successful switchers complete paths. Run `/assess-skills` to see your starting point.

### Q: Is this free?
**A**: Yes! Plugin is free. Some courses/platforms require paid subscriptions.

### Q: Can I use this offline?
**A**: Partially. Agents work offline, commands need internet for full resources.

### Q: How do I track progress?
**A**: Use `/assess-skills` quarterly. Plugin automatically tracks via hooks (if enabled).

### Q: What if I get stuck?
**A**: Ask the relevant agent with `@specialization-name question`. Join communities from `/find-resources`.

### Q: How often is content updated?
**A**: Based on 79 live roadmaps. Resources update every 24 hours.

### Q: Can I customize paths?
**A**: Yes! After `/learn-path`, customize weeks/projects based on your needs.
