# /learn-path

Choose your learning path and get a personalized roadmap for your career development.

## Usage

```
/learn-path
```

## Description

This command helps you select a learning specialization from the 7 major developer paths:

1. **Frontend Developer** - React, Vue, Angular, TypeScript, CSS
2. **Backend Developer** - Node.js, Python, Java, APIs, Databases
3. **DevOps Engineer** - Docker, Kubernetes, Cloud, Infrastructure
4. **AI/ML Engineer** - Machine Learning, Deep Learning, LLMs
5. **Mobile Developer** - iOS, Android, React Native, Flutter
6. **Data Engineer** - Data Pipelines, Warehouses, Big Data
7. **System Architect** - System Design, Scalability, Architecture

## What You'll Get

- A personalized learning roadmap for your chosen specialization
- Week-by-week learning plan
- Key technologies and tools to master
- Common challenges and how to overcome them
- Hands-on projects and practice resources
- Time estimates for each milestone
- Links to official documentation and communities

## Example Flow

1. Run `/learn-path`
2. Select your desired specialization
3. Receive a detailed learning plan with:
   - **Foundations** (Weeks 1-2): Core concepts
   - **Core Skills** (Weeks 3-6): Framework/language mastery
   - **Advanced Topics** (Weeks 7-10): Optimization and patterns
   - **Production Practice** (Weeks 11-12): Real-world deployment

## Tips

- Start with foundations even if you have prior experience
- Practice with real projects throughout your learning
- Join communities for peer support and networking
- Revisit advanced topics after gaining experience
- Track your progress and celebrate milestones
